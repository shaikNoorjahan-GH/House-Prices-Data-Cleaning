# House Prices Data Cleaning using Pandas

## 📌 Project Overview

This project demonstrates data cleaning and preprocessing techniques using the House Prices dataset with Python and Pandas. It covers data inspection, handling missing values, identifying duplicate records, descriptive statistics, and preparing the dataset for further analysis.

## 🎯 Objectives

- Load and inspect the dataset
- Explore dataset structure and summary statistics
- Identify and handle missing values
- Detect duplicate records
- Drop columns with excessive missing values
- Prepare a clean dataset for analysis and machine learning

## 📋 Data Cleaning Tasks Performed

- Loaded the dataset using Pandas
- Displayed the first and last records
- Checked dataset dimensions
- Listed all column names
- Verified data types
- Displayed dataset information using `info()`
- Generated descriptive statistics using `describe()`
- Identified duplicate rows
- Counted missing values
- Calculated missing value percentages
- Filled missing values using median, mode, and default values
- Replaced categorical missing values
- Dropped columns with more than 50% missing values
- Saved the cleaned dataset

## ▶️ How to Run

1. Clone this repository.
2. Open `House_Prices_Data_Cleaning.ipynb` using Google Colab or Jupyter Notebook.
3. Place `house_prices_raw_dataset.csv` in the same directory.
4. Run all notebook cells.
